# Project Name
Financial Tracker App

## Description
A C# console application to track income and expenses.


## Features
- Transaction Management: Easily add and track income and expense entries.
- Dynamic Summaries: View a complete breakdown of total income and expenses.
- Balance Calculation: Automatically calculates your current standing balance.
- Clean Formatting: Displays all financial data in a readable currency format.


## Getting Started

1. Clone the repository.
2. Open the `.sln` file in Visual Studio.
3. Build and Run.
